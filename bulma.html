Jag har använt ramverket Bulma CSS för aktuell undersida.

Bulma CSS är ett modernt open-source CSS-ramverk baserat på Flexbox, vilket gör det enkelt att bygga responsiva och användarvänliga webbgränssnitt utan att behöva skriva egen CSS-kod.

Till skillnad från utility-first-ramverk som Tailwind, använder Bulma ett klassiskt tillvägagångssätt där komponenter som buttons, forms, navbar och modals redan är färdiga och designade, vilket gör att utvecklare snabbt kan implementera vanliga element utan mycket konfiguration.

Bulma kännetecknas av sin enkla syntax och användning av fördefinierade klasser som is-primary, is-danger, is-small, och is-fullwidth. Med dessa klasser går det snabbt att justera design och färgscheman utan att skriva extra CSS.

Bulma är mobile-first. Responsiv design är inbyggd från början. Genom klasser som is-mobile, is-desktop, eller is-widescreen är det enkelt att skapa layoutanpassningar för olika skärmar.
