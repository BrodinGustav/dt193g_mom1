Vad är Bootstrap?

Bootstrap är ett open-source CSS-ramverk som riktar sig till responsiv mobile-first front-end webbutveckling.

Bootstrap.com
Bootstrap

Ramverket innehåller templates för HTML, CSS och JS.

Bootstrap

Bootstraps grundläggande layoutkomponent kallas för "container". Alla andra element på sidan placeras inuti denna container.

Bootstrap

När containern är på plats används sedan andra layoutkomponeneter i BootStrap för att implemenetera Flexbox-layoutsystem genom att definiera rader och kolumner.

Bootstrap

Container finns i storlekarna "fast bredd" eller "flytande bredd".

Bootstrap

Något jag uppskattar med Bootstrap är deras hantering av responsivitet genom inbyggda klasser.
